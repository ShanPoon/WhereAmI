# WhereAmI
## How to start

```
cd WhereAmI
roslaunch my_robot world.launch
```

Open another terminal
```
cd Project3_WhereAmI
roslaunch my_robot amcl.launch
```
To open teleop_twist_keyboard to control
<br/>
Open a new terminal

```
cd Project3_WhereAmI
rosrun teleop_twist_keyboard teleop_twist_keyboard.py
```
